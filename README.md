## 🎯 介紹
這是一款為 **Mstar 玩家** 設計的線上工具，  
可快速計算情侶等級升級所需的 **經驗值、M 幣**，  
並支援水果回收價計算（可回收 70% 價格）。

---

## 🧩 功能特色
- 📱 **手機與電腦皆可使用**  
- 🎮 支援「等級」與「經驗值」兩種輸入模式  
- 💰 自動計算所需 **M 幣**、**水果購買次數**、與 **實際花費**  
- 🍊 內建水果價位表（含回收比例）  
- ✨ 可愛動畫效果與深色主題介面  

---

## 🧮 使用說明
1. 選擇水果價位（或不計算水果）  
2. 選擇「當前」與「目標」等級／經驗值  
3. 點擊 **計算** 按鈕  
4. 即可顯示所需經驗、M 幣、購買數量與回收後實際花費
